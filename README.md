# Retail Shift Swap System

A Flask-based web application designed to simplify employee shift management, availability updates, and shift swap requests for retail teams.

## Project Overview

The Retail Shift Swap System helps employees request shift swaps and update availability while allowing managers to manage employees, create schedules, and review swap requests through a centralized dashboard.

## Key Features

- Employee login
- Manager login
- Employee availability updates
- Shift swap request submission
- Manager dashboard
- Employee management
- Schedule creation
- Swap request tracking
- SQLite database integration

## Technologies Used

- Python
- Flask
- SQLite
- HTML
- CSS
- JavaScript

## Business Value

This project supports retail workforce planning by reducing scheduling conflicts, improving communication between employees and managers, and centralizing shift management activities.
